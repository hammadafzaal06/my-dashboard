# My Dashboard

A responsive dashboard application built with React.js and Tailwind CSS. It includes authentication, reusable components, modern UI design, and state management. The project demonstrates real-world frontend development practices and clean code structure.

## 🚀 Features
- Responsive dashboard UI
- Authentication (local storage based)
- Reusable components
- Sidebar navigation
- Navbar with search and notifications
- Modern UI with Tailwind CSS

## 🛠 Tech Stack
- React.js
- Tailwind CSS
- Vite

## 📦 Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
